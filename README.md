## Características Principales
- **Micro-servicios:** La logica se creo bajo el modelo de microservicios para hacerlo mas modular y que cada parte sea independiente.
- **HTML como interfaz:** Creacion de una interfaz dinamica con HTML y JS.
- **Creacion de una API RESTful Completa**: Creación de un conjunto de *endpoints* (puntos de acceso) que cumplen con el ciclo de vida completo de los datos (CRUD), permitiendo la comunicación eficiente entre el frontend y el servidor.
